This is a basic project to circulate air inside of the greenhouse in my apartment to prevent mold and fungus flies. 
It uses an Arduino Nano to read a DHT11 temp/humidity sensor and display the readings to a 7-segment LCD. It also has a small DC motor to circulate the air by turning on and off every 15 minutes.
